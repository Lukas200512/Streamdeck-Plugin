# Security Policy

## Supported Versions

This plugin is early-stage; only the latest release and `main` branch receive fixes.

| Version      | Supported |
| ------------ | ----------|
| Latest (main)| ✅        |
| Older tags   | ❌        |

## Reporting a Vulnerability

- Please open a private security report via your Git hosting platform (Security Advisories) or send details to the maintainer/contact for this repository.
- Include reproduction steps, affected version/commit, and expected impact.
- You can expect an initial response within 7 business days.
